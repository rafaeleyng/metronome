# metronome

[Demo](https://rafaeleyng.github.io/metronome/)

## how to run

Requires NodeJS (see version in `.nvmrc` file)

```sh
git clone https://github.com/rafaeleyng/metronome.git
cd metronome
npm install # or yarn install
npm start # or yarn start
```
